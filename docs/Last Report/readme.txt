Para não haver conflitos de versões, cada um tem o seu tema num ficheiro auxiliar:

servidor.tex - capítulo do servidor
base.tex - capítulo da base de dados
etc etc

As alterações às partes de cada um são feitas no .tex correspondente e apenas é necessario fazer upload desse ficheiro para o repositório.

A compilação para ir vendo como está tudo a ficar é feita de forma normal, através do .tex principal (documento.tex)
